Структура для модулей такова: 
 
папка_со_скриптом/blocks/папка_с_модулем
 
Для создания простого модуля нужно создать файл: install.txt
 
Содержимое:
 
<name>New Module</name> // Название модуля
<desc>Do'h!</desc> // Описание модуля
<inst_folder>cats</inst_folder> // Папка с модулем
<author>Roman2861</author> // Автор скрипта
<version>0.0.1</version> // Версия скрипта
<image>1</image> // Есть ли логотип у модуля. Если есть, тогда есго местонахождение - папка_со_скриптом/blocks/папка_с_модулем/logo.png
<mysql>1</mysql> // Требует ли модуль mysql - запросов - об этом ниже
<block>1</block> // Требует ли модуль создания блока - об этом ниже
<page>1</page> // Требует ли модуль создания статической страницы(с текстом)  - об этом ниже
<need>login</need> // Зависимость от другого модуля. Если зависимостей нет, то пишите <need>no</need>
<pagev>1</pagev> // Будет ли ссылка на страницу в верхнем меню.
Mysql.
Все запросы для выполнения нужно написать и сохранить под названием db.sql
При удалении модуля создать файл del_db.sql с запросами для удаления сделанных изменений.
 
Блок.
Создаётся блок, и содержимое блока нужно написать в файл block.php
 
Страница.
Создаётся статическая страница, где можно написать любой текст. Содержимое описывается в page.html.  Если <page>2</page>, то содержимое страницы будет в modules/папка_с_модулем.
